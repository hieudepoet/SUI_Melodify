## 5. Core On-chain Objects

### Music

* Represents a song
* Ownable, tradable
* Has revenue pool
* Supports parent (remix)

### ListenCap

* Minted per listen
* Time-bounded
* Used for access & gamification

### RemixCap

* Grants permission to remix
* Enforces derivative rules

### SubscriptionCap

* Tiered access
* Anti-sybil primitive

### Badge

* Reputation & boosts
* Can be bound to Music or User
